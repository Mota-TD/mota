## 1. 功能概述

### 1.1 功能定位

AI智能新闻推送是摩塔平台的核心AI功能之一，旨在根据企业所属行业和业务领域，自动追踪、筛选并推送与企业相关的行业新闻、政策法规和市场动态，帮助企业及时掌握行业信息，做出更好的商业决策。

### 1.2 核心价值

| 价值维度 | 具体价值 | 量化指标 |
|----------|----------|----------|
| 信息获取效率 | 自动筛选相关新闻，减少人工搜索时间 | 节省80%信息搜索时间 |
| 政策敏感度 | 实时监控政策变化，及时预警 | 政策发布24小时内推送 |
| 决策支持 | 提供行业洞察，辅助商业决策 | 决策信息覆盖率提升60% |
| 知识沉淀 | 自动归档重要新闻，形成知识库 | 知识复用率提升40% |

### 1.3 目标用户

| 用户角色 | 使用场景 | 核心需求 |
|----------|----------|----------|
| 企业管理者 | 了解行业趋势和政策变化 | 高质量、高相关性的行业资讯 |
| 项目经理 | 获取项目相关的技术和市场信息 | 与项目领域匹配的专业资讯 |
| 业务人员 | 了解客户行业动态 | 客户行业的最新动态 |
| 研发人员 | 追踪技术发展趋势 | 技术领域的前沿资讯 |

### 3.5 NW-005 智能匹配

#### 3.5.1 功能描述

基于企业行业、业务领域和用户偏好，计算新闻与企业的相关性分数，实现精准推荐。

#### 3.5.2 匹配算法

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          智能匹配算法                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  匹配分数 = α × 行业匹配 + β × 关键词匹配 + γ × 语义相似度 + δ × 时效性    │
│                                                                             │
│  其中：                                                                     │
│  • α = 0.3  行业匹配权重                                                    │
│  • β = 0.25 关键词匹配权重                                                  │
│  • γ = 0.35 语义相似度权重                                                  │
│  • δ = 0.1  时效性权重                                                      │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  行业匹配                                                           │   │
│  │  • 主行业完全匹配: 100分                                            │   │
│  │  • 次行业匹配: 80分                                                 │   │
│  │  • 相关行业匹配: 60分                                               │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  关键词匹配                                                         │   │
│  │  • 核心关键词命中: 每个+20分                                        │   │
│  │  • 普通关键词命中: 每个+10分                                        │   │
│  │  • 最高100分                                                        │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  语义相似度                                                         │   │
│  │  • 使用向量数据库计算新闻与企业画像的语义相似度                     │   │
│  │  • 相似度 × 100 = 分数                                              │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  时效性                                                             │   │
│  │  • 1小时内: 100分                                                   │   │
│  │  • 24小时内: 80分                                                   │   │
│  │  • 7天内: 60分                                                      │   │
│  │  • 30天内: 40分                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

#### 3.7.3 推送渠道

| 渠道 | 说明 | 适用场景 |
