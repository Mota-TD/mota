Mota UI 组件库

一套现代化的企业级 UI 组件库，基于摩塔设计规范，支持 React 和 Vue 框架， 助力团队快速构建高质量的研发协同平台。

50+ 组件 TypeScript React & Vue MIT License

核心特性

丰富的组件

提供 50+ 高质量组件，覆盖表单、数据展示、反馈、导航等常见场景。

统一设计语言

基于摩塔设计规范，提供一致的视觉体验和交互模式。

框架无关

同时支持 React 和 Vue 框架，CSS 样式可独立使用。

TypeScript 支持

完整的类型定义，提供良好的开发体验和代码提示。

暗色主题

内置暗色主题支持，一键切换明暗模式。

响应式设计

组件自适应不同屏幕尺寸，支持移动端和桌面端。

安装

使用 npm

$ npm install @mota/ui

使用 yarn

$ yarn add @mota/ui

使用 pnpm

$ pnpm add @mota/ui

快速开始

React 项目

import '@mota/ui/dist/index.css';
import { Button, Input, Card } from '@mota/ui/react';

function App() {
  return (
    <Card title="欢迎使用 Mota UI">
      <Input placeholder="请输入内容" />
      <Button variant="primary">提交</Button>
    </Card>
  );
}

Vue 项目

import '@mota/ui/dist/index.css';
import MotaUI from '@mota/ui/vue';

const app = createApp(App);
app.use(MotaUI);
app.mount('#app');

// 在模板中使用
<template>
  <m-card title="欢迎使用 Mota UI">
    <m-input placeholder="请输入内容" />
    <m-button variant="primary">提交</m-button>
  </m-card>
</template>

Button 按钮

按钮用于触发一个操作，如提交表单、打开对话框等。

按钮类型

<Button variant="outline">边框按钮</Button>
<Button variant="ghost">幽灵按钮</Button>
<Button variant="danger">危险按钮</Button>

按钮尺寸

<Button size="small">小型按钮</Button>
<Button size="medium">默认按钮</Button>
<Button size="large">大型按钮</Button>

按钮形状

<Button>默认形状</Button>
<Button shape="round">圆角按钮</Button>
