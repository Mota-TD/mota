## 🏗️ 技术栈

### 前端技术
- **框架**: React 18 + TypeScript 5
- **UI组件**: Ant Design 5.x
- **状态管理**: Zustand
- **构建工具**: Vite
- **路由**: React Router v6

### 后端技术
- **框架**: Spring Boot 3.x + Java 17
- **ORM**: MyBatis-Plus
- **网关**: Spring Cloud Gateway

### AI能力
- **多模型支持**: GPT-4、Claude、通义千问
- **RAG检索**: 基于Milvus向量数据库
- **知识图谱**: Neo4j图数据库
