# ⚠️ 此项目已废弃

## 📢 重要通知

**mota-web-next** (Next.js 14) 项目已被废弃，不再维护。

## 🔄 迁移信息

项目已迁移至 **Vite + React** 架构，新项目位于：

```
📁 mota-web/
```

## 🎯 迁移原因

1. **性能提升**：Vite 提供更快的开发体验（冷启动 <1秒）
2. **架构简化**：传统 SPA 架构更适合当前项目需求
3. **维护性**：模块化架构更清晰，代码组织更合理
4. **开发体验**：极速的热更新和更灵活的配置

## 🚀 快速开始新项目

```bash
# 进入新项目目录
cd ../mota-web

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```
