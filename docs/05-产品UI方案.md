## 11. 无障碍设计

### 11.1 无障碍规范

| 规范项 | 要求 | 实现方式 |
|--------|------|----------|
| 颜色对比度 | WCAG AA级别 (4.5:1) | 使用对比度检查工具验证 |
| 键盘导航 | 所有功能可键盘操作 | Tab顺序、焦点样式 |
| 屏幕阅读器 | 语义化HTML | ARIA标签、alt文本 |
| 字体大小 | 支持放大200% | rem单位、响应式布局 |
| 动画 | 可禁用动画 | prefers-reduced-motion |
