### 1.2 组件命名规范

- 使用 PascalCase 命名组件
- 组件名称应清晰表达其功能
- 避免使用缩写（除非是通用缩写）

## 2. 基础组件

### 2.1 按钮 Button

#### 组件属性

| 属性 | 类型 | 默认值 | 说明 |
|-----|------|-------|------|
| type | string | 'secondary' | 按钮类型：primary/secondary/outline/ghost/danger/link |
| size | string | 'medium' | 按钮尺寸：small/medium/large |
| disabled | boolean | false | 是否禁用 |
| loading | boolean | false | 是否加载中 |
| icon | ReactNode | - | 按钮图标 |
| block | boolean | false | 是否块级按钮 |
| onClick | function | - | 点击事件 |

#### 样式规范

**Primary Button**
```css
.btn-primary {
    background: linear-gradient(135deg, #2b7de9 0%, #1a6dd6 100%);
    color: #ffffff;
    border: none;
    padding: 10px 16px;
    border-radius: 8px;
    font-size: 14px;
    font-weight: 500;
    cursor: pointer;
    transition: all 0.15s ease;
}

.btn-primary:hover {
    background: linear-gradient(135deg, #1a6dd6 0%, #1560c0 100%);
    transform: translateY(-1px);
    box-shadow: 0 4px 12px rgba(43, 125, 233, 0.3);
}

.btn-primary:active {
    transform: translateY(0);
}

.btn-primary:disabled {
    opacity: 0.5;
    cursor: not-allowed;
    transform: none;
}
```

**Secondary Button**
```css
.btn-secondary {
    background: #ffffff;
    color: #1f2329;
    border: 1px solid #dee0e3;
    padding: 10px 16px;
    border-radius: 8px;
    font-size: 14px;
    font-weight: 500;
    cursor: pointer;
    transition: all 0.15s ease;
}

.btn-secondary:hover {
    border-color: #2b7de9;
    color: #2b7de9;
}
```

**按钮尺寸**
```css
.btn-sm {
    padding: 6px 12px;
    font-size: 13px;
}

.btn-md {
    padding: 10px 16px;
    font-size: 14px;
}

.btn-lg {
    padding: 12px 24px;
    font-size: 15px;
}
```

#### 使用示例

```html
<!-- 主要按钮 -->
<button class="btn btn-primary">确认</button>

<!-- 次要按钮 -->
<button class="btn btn-secondary">取消</button>

<!-- 带图标按钮 -->
<button class="btn btn-primary">
    <svg>...</svg>
    <span>新建项目</span>
</button>

<!-- 加载状态 -->
<button class="btn btn-primary" disabled>
    <span class="loading"></span>
    <span>提交中...</span>
</button>
```

### 2.3 标签 Tag

| 属性 | 类型 | 默认值 | 说明 |
|-----|------|-------|------|

### 2.5 头像 Avatar

#### 组件属性

| 属性 | 类型 | 默认值 | 说明 |

```css
.avatar {
    width: 32px;
    height: 32px;

    height: 100%;

.avatar-sm { width: 24px; height: 24px; font-size: 12px; }
.avatar-md { width: 32px; height: 32px; font-size: 14px; }
.avatar-lg { width: 48px; height: 48px; font-size: 18px; }
.avatar-xl { width: 64px; height: 64px; font-size: 24px; }
```


## 8. 组件使用指南

### 8.1 组件组合原则

1. **保持一致性**：相同功能使用相同组件
2. **合理嵌套**：避免过深的组件嵌套
3. **语义化使用**：根据场景选择合适的组件
4. **性能优化**：避免不必要的组件渲染

### 8.2 组件状态管理

- 使用统一的状态管理方案
- 组件内部状态与全局状态分离
- 合理使用受控与非受控组件

### 8.3 组件可访问性

- 提供键盘导航支持
- 添加适当的 ARIA 属性
- 确保足够的颜色对比度
- 支持屏幕阅读器
