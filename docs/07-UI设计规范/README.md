# 摩塔 Mota - UI 设计规范文档

## 📋 文档概述

本目录包含摩塔 Mota 产品的完整 UI 设计规范文档，为设计师和开发人员提供统一的设计标准和实施指南。

```

├── 03-页面设计规范.md        # 页面布局与模板规范
└── 04-交互设计规范.md        # 交互设计与动效规范
```

## 📖 文档内容

### 1. UI 设计方案 ([01-UI设计方案.md](./01-UI设计方案.md))

整体设计方案文档，包含：

- **设计原则**：清晰性、一致性、高效性、友好性、专业性
- **色彩系统**：主色调、功能色、中性色、扩展色板
- **字体系统**：字体家族、字号规范、字重规范
- **间距系统**：基础间距单位、组件间距规范
- **圆角系统**：各级圆角值定义
- **阴影系统**：阴影层级与使用场景
- **动效系统**：过渡时间、缓动函数、常用动画
- **图标系统**：图标规范与使用指南
- **布局系统**：页面布局结构、栅格系统
- **响应式设计**：断点定义、适配策略

### 2. 组件库规范 ([02-组件库规范.md](./02-组件库规范.md))

详细的组件库规范，包含：

- **基础组件**：Button、Icon、Tag、Badge、Avatar
- **表单组件**：Input、Select、Checkbox、Radio、Switch、DatePicker
- **数据展示**：Table、Card、List、Tooltip
- **反馈组件**：Modal、Toast、Loading、Progress、Empty、Skeleton
- **导航组件**：Menu、Tabs、Breadcrumb、Pagination、Dropdown
- **业务组件**：IssueCard、ProjectCard、StatCard、ActivityItem

### 3. 页面设计规范 ([03-页面设计规范.md](./03-页面设计规范.md))

页面级别的设计规范，包含：

- **营销页面**：首页、登录页、注册页
- **控制台页面**：工作台、项目列表、事项列表、事项详情
- **功能页面**：看板、迭代详情、知识库、设置
- **通用元素**：页面标题区、面包屑、筛选工具栏、分页器
- **响应式适配**：各断点布局调整策略

### 4. 交互设计规范 ([04-交互设计规范.md](./04-交互设计规范.md))

交互设计与动效规范，包含：

- **鼠标交互**：点击、悬停、拖拽、右键菜单
- **键盘交互**：快捷键、焦点管理、Tab 导航
- **表单交互**：输入、选择、验证、提交
- **导航交互**：页面导航、弹窗、下拉菜单
- **数据交互**：加载、刷新、操作、批量处理
- **反馈交互**：消息提示、通知、进度、空状态
- **动效规范**：动效原则、时长、缓动函数
- **无障碍交互**：键盘可访问性、屏幕阅读器支持
- **移动端交互**：手势、触摸反馈、适配

## 🎨 设计系统速查

### 主色调

| 名称 | 色值 | 用途 |
|-----|------|-----|
| Primary | `#2b7de9` | 主要按钮、链接、选中状态 |
| Primary Hover | `#1a6dd6` | 主色悬停状态 |
| Primary Light | `#e6f0fc` | 主色浅色背景 |

### 功能色

| 名称 | 色值 | 用途 |
|-----|------|-----|
| Success | `#10b981` | 成功状态 |
| Warning | `#f59e0b` | 警告状态 |
| Error | `#ef4444` | 错误状态 |
| Info | `#3b82f6` | 信息提示 |

### 字号规范

| 级别 | 字号 | 用途 |
|-----|------|-----|
| H1 | 28px | 页面主标题 |
| H2 | 24px | 区块标题 |
| H3 | 18px | 卡片标题 |
| Body | 14px | 正文内容 |
| Small | 12px | 辅助文字 |

### 间距规范

| 级别 | 数值 | 用途 |
|-----|------|-----|
| xs | 4px | 紧凑间距 |
| sm | 8px | 小间距 |
| md | 12px | 中间距 |
| base | 16px | 标准间距 |
| lg | 20px | 大间距 |
| xl | 24px | 页面边距 |

### 圆角规范

| 级别 | 数值 | 用途 |
|-----|------|-----|
| sm | 4px | 标签、徽章 |
| md | 8px | 按钮、输入框 |
| lg | 12px | 卡片、弹窗 |

## 🔧 使用指南

### 设计师

1. 阅读 [01-UI设计方案.md](./01-UI设计方案.md) 了解整体设计系统
2. 参考 [02-组件库规范.md](./02-组件库规范.md) 使用标准组件
3. 按照 [03-页面设计规范.md](./03-页面设计规范.md) 设计页面布局
4. 遵循 [04-交互设计规范.md](./04-交互设计规范.md) 设计交互效果

### 开发人员

1. 参考 CSS 变量定义实现设计系统
2. 按照组件规范开发 UI 组件
3. 遵循页面模板实现页面布局
4. 实现规范中定义的交互效果

### 产品经理

1. 了解设计规范确保需求与设计一致
2. 参考页面模板规划功能布局
3. 理解交互规范评估用户体验
