## 2. 营销页面设计

#### 页面结构

```

│                        公告栏 (可关闭)                            │
├─────────────────────────────────────────────────────────────────┤

├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│                     Hero 区域                                    │
│           主标题 + 副标题 + CTA按钮 + 数据统计                     │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│                   产品功能展示区                                  │
│              项目协同 / 知识管理 / 效能洞察                        │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│                   角色价值展示区                                  │
│         管理者 / 项目经理 / 产品经理 / 测试 / 开发                  │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│                   工作流程展示区                                  │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│                   客户案例展示区                                  │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│                   CTA 行动号召区                                  │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                        页脚                                      │
│           产品链接 / 解决方案 / 支持 / 公司 / 社交媒体              │
└─────────────────────────────────────────────────────────────────┘
```

#### 设计规范

**导航栏**
- 高度：64px
- 背景：白色/透明（滚动后变白色）
- Logo 尺寸：36px × 36px
- 导航项间距：32px
- 按钮样式：Ghost + Primary

**Hero 区域**
- 最小高度：600px
- 背景：渐变色 + 装饰图案
- 主标题：48px - 56px，加粗
- 副标题：18px - 20px，次要色
- CTA 按钮：大尺寸，主色调

**功能卡片**
- 卡片尺寸：等宽四列
- 卡片间距：24px
- 图标尺寸：48px
- 悬停效果：上移 + 阴影

## 5. 响应式页面设计

### 5.1 断点适配

| 断点 | 布局调整 |
|-----|---------|
| ≥ 1400px | 完整布局，侧边栏展开 |
| 1200px - 1399px | 内容区域缩窄 |
| 992px - 1199px | 侧边栏可折叠 |
| 768px - 991px | 侧边栏隐藏，汉堡菜单 |
| < 768px | 移动端布局，单列显示 |

- 搜索改为图标触发

- 表格支持横向滚动
- 卡片改为单列
- 表单全宽显示

**交互适配**

- 支持手势操作
- 简化操作步骤
