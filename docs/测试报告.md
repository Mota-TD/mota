## 🔧 发现并修复的问题

### 问题 1: 登录页面忘记密码链接无效

**严重程度**: 🟡 中等  
**文件位置**: `mota-user/src/pages/auth/Login/index.tsx`  
**问题描述**: 忘记密码链接指向 `/forgot-password` 路由，但该路由未在路由配置中定义，点击会导致404错误。

**修复方案**: 将链接改为显示提示信息 "请联系管理员重置密码"

**修复代码**:
```tsx
<a
  href="#"
  className={styles.forgotLink}
  onClick={(e) => {
    e.preventDefault()
    message.info('请联系管理员重置密码')
  }}
>
  忘记密码？
</a>
```

---

### 问题 2: 项目详情页事项导航参数错误

**严重程度**: 🔴 高  
**文件位置**: `mota-user/src/pages/project-detail/index.tsx`  
**问题描述**: 事项列表中点击事项key时，使用 `key` 参数进行导航，但路由需要的是 `id` 参数，导致无法正确跳转到事项详情页。

**修复方案**: 将导航参数从 `key` 改为 `record.id`

**修复代码**:
```tsx
render: (key: string, record: Issue) => (
  <a onClick={() => navigate(`/issues/${record.id}`)}>{key}</a>
)
```

---

### 问题 3: 控制台布局快速创建菜单链接无效

**严重程度**: 🟡 中等  
**文件位置**: `mota-user/src/layouts/ConsoleLayout/index.tsx`  
**问题描述**: 快速创建菜单中的链接指向 `/projects/new`、`/issues/new`、`/wiki/new` 等路由，但这些路由未定义。

**修复方案**: 将链接改为导航到对应的列表页面

**修复代码**:
```tsx
const quickCreateItems = [
  { key: 'project', icon: <FolderAddOutlined />, label: '新建项目', path: '/projects' },
  { key: 'issue', icon: <PlusCircleOutlined />, label: '新建任务', path: '/issues' },
  { key: 'wiki', icon: <FileAddOutlined />, label: '新建文档', path: '/wiki' },
]
```

### 问题 4: 看板页面状态列与数据不一致

**严重程度**: 🟡 中等  
**文件位置**: `mota-user/src/pages/kanban/index.tsx`  
**问题描述**: 看板定义了 "review"（待评审）状态列，但 mock 数据中有 "testing" 状态的事项，没有 "review" 状态的事项，导致测试中的事项无法正确显示。

**修复方案**: 将 "review" 列改为 "testing"（测试中）列

**修复代码**:
```tsx
const [columns, setColumns] = useState<KanbanColumn[]>([
  { id: 'open', title: '待处理', status: 'open', issues: [], limit: 10 },
  { id: 'in_progress', title: '进行中', status: 'in_progress', issues: [], limit: 5 },
  { id: 'testing', title: '测试中', status: 'testing', issues: [], limit: 5 },
  { id: 'done', title: '已完成', status: 'done', issues: [] }
])
```

const quickActions = [
