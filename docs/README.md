## 🎯 产品简介

**摩塔（Mota）** 是一款AI驱动的企业级项目管理与知识协作平台，致力于帮助企业实现：

- 🧠 **知识资产化** - 将分散的企业知识转化为可检索、可复用的智能资产
- 🤝 **智能协作** - AI驱动的任务分解、智能分工、进度追踪
- 📊 **数据驱动决策** - 实时数据分析、AI洞察、智能报告

---

## 🏗️ 技术栈

### 前端
- React 18 + TypeScript 5
- Ant Design 5.x
- Zustand 状态管理
- Vite 构建工具

### 后端
- Spring Boot 3.x + Java 17
- MyBatis-Plus
- Spring Cloud Gateway

### 数据存储
- MySQL 8.x - 主数据库
- Redis 7.x - 缓存
- Milvus 2.x - 向量数据库
- Neo4j 5.x - 图数据库
- Elasticsearch 8.x - 搜索引擎

### AI能力
- 多模型支持：GPT-4、Claude、通义千问
- RAG检索增强生成
- 知识图谱
