## 技术架构

### 前端技术栈
- **框架**: uni-app + Vue 3 + TypeScript
- **构建**: Vite 5
- **状态**: Pinia
- **UI**: uni-ui
- **样式**: SCSS

### 核心特性
- ✅ 跨平台支持（iOS/Android/HarmonyOS/H5/小程序）
- ✅ TypeScript严格模式
- ✅ 模块化架构
- ✅ 流式AI通信
- ✅ 语音交互
- ✅ 智能提醒
- ✅ 离线缓存

### 后端集成
- **服务**: mota-service（Spring Boot微服务）
- **通信**: RESTful API + WebSocket
- **认证**: JWT Token
- **AI**: 集成AI服务接口

## 开发规范

### 代码规范
- 使用TypeScript严格模式
- 遵循Vue 3 Composition API
- 统一的错误处理
- 完整的类型定义

### 命名规范
- 文件名：kebab-case
- 组件名：PascalCase
- 变量名：camelCase
- 常量名：UPPER_SNAKE_CASE

### 目录规范
```
src/
├── pages/          # 页面（按功能模块组织）
├── components/     # 组件（按类型组织）
├── core/          # 核心服务（按业务模块组织）
├── stores/        # 状态管理
├── styles/        # 全局样式
└── static/        # 静态资源
```

## 常见问题

### 1. 如何启动项目？
参考 [快速启动指南](../QUICK_START.md)

### 2. 如何添加新功能？
1. 在 `core/` 下创建服务模块
2. 在 `pages/` 下创建页面
3. 在 `pages.json` 中注册路由
4. 更新文档

### 3. 如何调试？
- H5: 浏览器开发者工具
- 小程序: 微信开发者工具
- App: HBuilderX真机调试

### 4. 如何构建发布？
```bash
pnpm build:h5           # H5版本
pnpm build:mp-weixin    # 微信小程序
pnpm build:app-android  # Android App
pnpm build:app-ios      # iOS App
```
