## 快速开始

### 安装依赖
```bash
cd mota-app
pnpm install
```

### 开发运行
```bash
# H5
pnpm dev:h5

# 微信小程序
pnpm dev:mp-weixin

# Android
pnpm dev:app-android

# iOS
pnpm dev:app-ios
```

### 生产构建
```bash
# H5
pnpm build:h5

# 微信小程序
pnpm build:mp-weixin

# App
pnpm build:app-android
pnpm build:app-ios
```

## 注意事项

1. **TypeScript 错误**: 运行 `pnpm install` 安装依赖后会自动解决类型定义问题。

2. **环境变量**: 需要根据实际后端服务地址修改 `.env.development` 和 `.env.production`。

3. **静态资源**: 需要准备 logo 等静态资源文件。

4. **后端对接**: 需要确保 mota-service 后端服务已启动并正常运行。

5. **推送服务**: 消息推送功能需要配置相应的推送服务（如 UniPush）。
